# TOTP App

Небольшой десктопный менеджер TOTP-кодов
## Технологии
- Tauri v2
- React 19 + TypeScript
- Vite

## Установка и запуск

### Требования
- Node.js (LTS)
- Rust toolchain
- Предустановки Tauri для вашей ОС: https://tauri.app/v2/guides/getting-started/prerequisites/

### Установка
```bash
npm install
```

### Запуск (dev)
```bash
npm run tauri dev
```

### Сборка
```bash
npm run build
npm run tauri build
```

### Проверка линта
```bash
npm run lint
```
